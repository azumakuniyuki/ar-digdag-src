Ansible Role: ar-digdag-src

- Install digdag

--------------------------------------------------------------------------------
No requiremetns.

--------------------------------------------------------------------------------

installed packages on each supported platform.

Example Playbook

  roles:
     - azumakuniyuki.ar-digdag-src
